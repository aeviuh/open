-------------------------------------------------------------
一、导航与跳转类
Ctrl + B：跳转到变量、方法、类的定义处（最常用之一，点击引用快速溯源）。
Ctrl + N：快速查找类（输入类名直接定位，支持模糊匹配）。
Ctrl + Shift + N：快速查找文件（包括非 Java 文件，如 XML、配置文件等）。
Ctrl + G：跳转到指定行号（输入行数回车，快速定位代码位置）。
Ctrl + E：显示最近打开的文件列表（快速切换刚编辑过的文件）。
Alt + ←/→：在编辑过的文件之间前后切换（类似浏览器的前进 / 后退）。
Ctrl + Alt + B：跳转到接口的实现类（如接口有多个实现，会列出选择）。
Ctrl + Shift + Backspace：跳转到上一次编辑的位置（快速回退到修改处）。
二、代码编辑类
Ctrl + C / Ctrl + X / Ctrl + V：复制、剪切、粘贴（基础操作）。
Ctrl + D：复制当前行到下一行（无需选中，直接复制整行）。
Ctrl + Y：删除当前行（比选中删除更高效）。
Ctrl + W：逐步选中代码块（从光标处开始，多次按可扩大选中范围，如单词→行→方法）。
Ctrl + Shift + W：逐步取消选中（与 Ctrl + W 相反）。
Ctrl + Z / Ctrl + Shift + Z：撤销 / 反撤销（恢复误操作）。
Ctrl + Shift + U：切换选中文字的大小写（快速处理变量名或常量）。
Tab / Shift + Tab：缩进 / 取消缩进（选中多行时批量调整）。
三、代码生成与补全类
Ctrl + Space：基础代码补全（如变量名、方法名提示，默认可能与输入法冲突，建议修改）。
Ctrl + Shift + Space：智能代码补全（根据上下文提示更精准的内容，如方法参数）。
Alt + Insert：生成代码（如 getter/setter、构造方法、toString 等，光标在类中时使用）。
Ctrl + O：重写父类方法（快速选择要重写的方法）。
Ctrl + I：实现接口方法（接口的实现类中快速生成未实现的方法）。
Ctrl + J：插入代码模板（如 sout 生成 System.out.println()，fori 生成 for 循环等）。
四、搜索与替换类
Ctrl + F：在当前文件内搜索（支持正则表达式）。
Ctrl + R：在当前文件内替换（配合 Ctrl + F 使用，替换匹配内容）。
Ctrl + Shift + F：全局搜索（在整个项目中搜索指定内容）。
Ctrl + Shift + R：全局替换（在整个项目中替换指定内容，谨慎使用）。
Shift + Shift：全局搜索一切（文件、类、方法、设置等，万能搜索）。
五、重构类（代码优化常用）
Shift + F6：重命名（变量、方法、类等，会自动修改所有引用处，安全高效）。
F5：复制文件 / 文件夹（重构时复制并调整引用）。
F6：移动文件 / 文件夹（调整类的位置，自动更新引用）。
Ctrl + Alt + M：抽取方法（将选中代码块抽取为一个方法，优化代码结构）。
Ctrl + Alt + V：抽取变量（将复杂表达式抽取为一个变量，提高可读性）。
六、窗口与视图操作
Alt + 1：显示 / 隐藏项目结构窗口（快速切换项目视图）。
Alt + 4：显示 / 隐藏控制台（查看程序输出或错误信息）。
Ctrl + Shift + F12：最大化编辑窗口（隐藏所有侧边栏，专注代码编辑）。
Ctrl + Tab：切换打开的文件标签（类似浏览器切换标签，按 Tab 时可选择）。
