HTTP协议介绍:
    HTTP协议:  HyperText Transfer Protocol（超文本传输协议)
    基于TCP/IP协议的应用层协议

HTTP请求报文:
    get:
        请求行:
            请求方法: get
            请求路径: /
            协议版本: HTTP/1.1
        请求头:
            键值对
        空行
        请求体:
            一般不写
    post:
        请求行:
            请求方法: post
            请求路径: /
            协议版本: HTTP/1.1
        请求头:
            键值对
        空行
        请求体:
            键值对
            键值对
            键值对

HTTP响应报文:
    响应行:
        协议版本: HTTP/1.1
        状态码: 200
        状态码描述: OK
    响应头:
        键值对
    空行
    响应体:
        一般是html文件
HTTP状态码:
    1xx: 信息响应
    2xx: 成功响应
    3xx: 重定向响应
    4xx: 客户端错误响应
    5xx: 服务器错误响应
    常用的状态码:
        200: OK # 成功
        400: Bad Request # 客户端请求错误
        404: Not Found # 资源不存在
        500: Internal Server Error # 服务器内部错误
