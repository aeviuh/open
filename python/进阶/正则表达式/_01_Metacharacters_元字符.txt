#Metacharacters_元字符
. # 匹配除了 换行符 的任意 单个字符

* # 修饰前面一个字符 匹配 0 次或多次
*? # 非贪婪模式 修饰前面一个字符 匹配 0 次或多次

+ # 修饰前面一个字符 匹配 1 次或多次
+? # 非贪婪模式 修饰前面一个字符 匹配 1 次或多次

? # 修饰前面一个字符 匹配 0 次或 1 次
{0,1} # 修饰前面一个字符 匹配 0 次或 1 次
?? # 非贪婪模式 修饰前面一个字符 匹配 0 次或 1 次

\ # 转义字符

\. # 匹配 点 本身
\* # 匹配 星号 本身
\+ # 匹配 加号 本身
\? # 匹配 问号 本身
\^ # 匹配 脱字符 本身
\$ # 匹配 美元符号 本身
\# # 匹配 井号 本身

\d # 匹配 数字
\D # 匹配 非数字
\w # 匹配 字母 数字 下划线
\W # 匹配 非字母 数字 下划线
\s # 匹配 空白字符
\S # 匹配 非空白字符


[] # 匹配 方括号 中的 任意 单个字符
[368] # 匹配 3 6 8 中的 任意 单个字符
[.+*?] # 匹配 点 加号 星号 问号 中的 任意 单个字符
[0-9] # 匹配 0-9 中的 任意 单个字符
[^abc] # 匹配 非(a b c) 中的 任意 单个字符,abc是一起的
[^\d] # 匹配 非数字 中的 任意 单个字符

^ # 匹配 字符串的开头, 多行模式下 匹配 每一行的开头,单行模式下 匹配 整个文本的开头
^\d # 匹配 以数字开头 的 字符串

$ # 匹配 字符串的结尾, 多行模式下 匹配 每一行的结尾,单行模式下 匹配 整个文本的结尾
\d$ # 匹配 以数字结尾 的 字符串

{x,y} # 匹配前面的字符 至少 x 次 至多 y 次
{x} # 匹配前面的字符 x 次
{x,} # 匹配前面的字符 至少 x 次 至多 无穷多次
{0.1} # 匹配前面的字符 0 次 至多 1 次

| # 匹配 | 左边 或 右边 的 任意 单个字符
() # 分组, 直接提取 匹配的 子字符串
